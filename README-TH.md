## วิธีอัปเดตบน GitHub

1. ลบไฟล์เว็บเวอร์ชันเดิม
2. แตก ZIP นี้
3. อัปโหลดไฟล์ทั้งหมดพร้อมกันไว้ที่หน้าแรกของ Repository
4. Commit changes
5. รอ 1–2 นาที แล้วกด Ctrl + F5
